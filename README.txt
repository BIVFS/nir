Для сборки (make) испольовать находящийся в текущей директории Makefile

При запуске программы будет предложен выбор режима(обучение или классификация)

Для обучения классификатора необходимо в файле source.txt указать ссылки на веб-страницы, каждая с новой строки, запустить программу и выбрать соответствуюий режим

Для классификации необходимо запустить программу, выбрать соответствующий режим и ввести ссылку на веб-страницу, чтобы закончить классификацию необходимо ввести 0
